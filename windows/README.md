# Windows bootstrap

1. Install Git (provides Git Bash). Example:
```powershell
winget install --id Git.Git
```
2. Clone this repo 
```bash
git clone git@github.com:nudabagana/machine-bootstrap.git
```
3. Open Git Bash and go into the Windows folder
```bash
cd machine-bootstrap/windows
```
4. Run setup script
```bash
./setup.sh
```
